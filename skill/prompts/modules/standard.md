# 课程标准生成

## 依赖
- 需引用课时分配方案的周次安排和学时分配数据

## 输出格式
- docx 格式，输出到 `{课程名}/` 目录
- 文件名：`{课程名}_课程标准_{timestamp}.docx`

## 内容结构

### 1. 课程基本信息
- 课程名称、课程编码、适用专业
- 学时/学分、课程类型、开课学期
- 先修课程、后续课程

### 2. 课程性质与任务
- 课程定位（专业必修/选修）
- 课程任务描述（100-200字）
- 与前后课程的关系说明

### 3. 课程目标
- 知识目标（3-4条）
- 技能目标（3-4条）
- 素养目标（2-3条）

### 4. 课程内容与要求
- 按项目/模块/章节组织
- 每个内容单元包括：名称、学时、知识要求、技能要求、教学重点与难点
- 使用表格形式呈现

### 5. 课程实施建议
- 教学方法建议
- 教学条件与资源
- 教材选用建议

### 6. 考核评价
- 考核方式（过程性考核+终结性考核）
- 成绩构成（平时30%+期末70%或类似比例）
- 评价标准说明
