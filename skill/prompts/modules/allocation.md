# 课时分配方案生成

## 输出格式
- 输出到 `{课程名}/` 目录
- 文件名：`{课程名}_课时分配表_{timestamp}.docx`

## 内容结构

### 1. 基本信息表
- 课程名称、总学时、周学时（2）、周数（总学时/2）
- 课程类型（B类/C类）、适用专业、年级

### 2. 周次分配表
- 周次列（第1周 ~ 第N周）
- 项目/模块名称
- 学时
- 教学内容概要（每行50-80字描述该周的教学内容要点）
- 教学方式（讲授/实践/混合）

### 3. 每章课时分配表（教材详细分配）
- 章序号、章名称
- 理论学时、实践学时
- 合计学时
- 字数预算（理论学时×45分钟×210字/分钟×系数0.6）

### 4. 说明
- 考核方式说明
- 教学资源需求
