# 教学实施Guide生成

## 依赖
- 需引用课时分配方案的周次安排

## 输出格式
- docx 格式，输出到 `{课程名}/` 目录
- 文件名：`{课程名}_实施方案_{timestamp}.docx`

## 内容结构

### 1. 课程概要
- 课程名称、总学时、周次安排
- 课程目标回顾

### 2. 教学进度安排
- 周次、教学内容、教学方式
- 对应的教材章节

### 3. 教学方法设计
- 整体教学策略说明
- 各教学单元采用的具体教学方法

### 4. 课程考核方案
- 考核内容与权重
- 评分标准

### 5. 教学资源
- 教材、参考书、实训设备、软件平台
